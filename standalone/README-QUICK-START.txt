CT Food Outlook - Quick Start Guide
====================================

FOR USERS:
----------

Windows:
1. Double-click "outlook-auto-attach.exe"
2. The app runs in the system tray (bottom-right corner)
3. Done! It's now monitoring your Downloads folder

macOS:
1. Double-click "CT Food Outlook.app"
2. The app runs in the menu bar (top-right corner)
3. Done! It's now monitoring your Downloads folder

HOW IT WORKS:
-------------

- The app runs in the background (system tray/menu bar)
- It automatically watches your Downloads folder
- When you download a file containing "Orderbekräftelse" or "Orderbekr"
- It automatically:
  1. Creates a copy with a clean name in Desktop/businessnxtdocs/
  2. Opens Microsoft Outlook with the file attached

NO SETUP NEEDED:
----------------

- Just double-click to start
- No installation required (though Windows installer is available)
- No configuration needed
- Works immediately

TO QUIT:
--------

Windows: Right-click the system tray icon → Quit
macOS: Click the menu bar icon → Quit

TO VIEW LOGS:
-------------

Windows: Right-click system tray icon → View Log
macOS: Click menu bar icon → View Log

Logs are saved to:
- Windows: %USERPROFILE%\AppData\Local\OutlookAutoAttach\
- macOS: ~/Library/Logs/OutlookAutoAttach/

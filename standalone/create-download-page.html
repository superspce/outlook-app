Installation Instructions

macOS:

  1. Download the .dmg file
  2. Double-click to open it
  3. Drag "CT Food Outlook.app" to the Applications folder
  4. Open Applications and launch the app
  5. The app will appear in your menu bar (top-right)

Windows:

  1. Download the .zip file
  2. Extract the zip file
  3. Right-click "Install.bat" → "Run as Administrator"
  4. The app will install and launch automatically
  5. The app will appear in your system tray (bottom-right)

How It Works

The app runs in the background and automatically:

  • Monitors your Downloads folder
  • Detects files containing "Orderbekräftelse"
  • Opens Microsoft Outlook with the file attached

No configuration needed - just install and it works!
